# Herbalife Sales Manager

A full-stack web application to digitalize the manual notebook-based tracking of sales, stock, and memberships for Herbalife distributors — saving time and eliminating errors.

**Live Demo:** [sales-manage-mauve.vercel.app](https://sales-manage-mauve.vercel.app/)

## Features

- Sales & revenue tracking
- Inventory management with low-stock alerts
- Customer & membership records
- Center sales module
- Dashboard analytics with volume point tracking
- Printable period reports for profits and revenue

## Tech Stack

- **Frontend:** Next.js 14 (App Router), Tailwind CSS, shadcn/ui
- **Backend:** Supabase (PostgreSQL + Auth)
- **Language:** TypeScript
- **Forms:** react-hook-form + Zod
